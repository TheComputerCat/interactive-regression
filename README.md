# Visualizador Interactivo de Puntos y Regresión

**Demo online:** [https://thecomputercat.github.io/interactive-regression/](https://thecomputercat.github.io/interactive-regression/)

Permite agregar, mover y eliminar puntos en un canvas, dibujar líneas personalizadas y ver de manera interactiva la regresión lineal, las medias, los residuales y la regresión cuantílica para el cuantil τ = 0.5.

## Cómo usar

1. Clic izquierdo: agregar un punto.
2. Arrastrar un punto: moverlo.
3. Clic derecho sobre un punto: borrarlo.
4. Para dibujar una línea, clic en dos puntos.
5. Casillas: mostrar/ocultar regresión lineal, media y residuales.
6. Casilla "Regresión cuantil τ = 0.5": mostrar/ocultar la recta de regresión cuantílica(Beta).
7. Botón "Borrar": eliminar todo.
